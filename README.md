# 规则

给出单词释义，另外将答案糢糊化，写出正确的单词即可。

![image](./snapshot/guess-word.gif)

# using

> M-: (guess-word)

## keybinding

* `<return>` 检查输入是否正确, 如果错误情况下，清除所有输入，再按下 `<return>` 会显示正确答案
* `C-r` 切换词库
* `M-x guess-word-add-dictionary-path` 添加自定义字典
* `M-x guess-word-delete-dictionary-path` 删除某一个词典
